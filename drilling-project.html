Optimization of Bi-Directional Oil and Gas Drilling Equipment Using MATLAB

Final Year Project • All Nations University, Koforidua • October 2022

Project Overview

In oil and gas drilling operations, downhole vibrations are a major cause of poor wellbore quality, equipment wear, and increased operational costs. This project focused on developing a simulation-based control system to actively reduce these vibrations and improve drill bit positioning accuracy.

Using MATLAB and Simulink, a Linear Quadratic Regulator (LQR) controller was designed and tested to stabilize the drilling system. The goal was to demonstrate that advanced control strategies could be developed and validated through simulation before expensive field implementation.

Aim

To develop a MATLAB/Simulink model capable of controlling drilling string equipment and significantly improving the accuracy of drill bit positioning by suppressing downhole vibrations.

Objectives

  • Design a realistic dynamic model of the drill string and bit in Simulink
  • Develop sensor signal processing modules to filter noise from downhole vibration data
  • Design and optimally tune a Linear Quadratic Regulator (LQR) controller
  • Evaluate the controller’s performance in reducing oscillations under realistic disturbance conditions

Methodology

A high-fidelity drill string and bit model was developed using mass-spring-damper principles in Simulink. Realistic sensor noise was introduced using signal processing techniques. An LQR controller was designed using state-space methods and tuned through iterative simulation. Root Locus analysis was used to study system stability. A high-torque DC motor model was integrated to represent actuation. Multiple test scenarios were created to evaluate how effectively the controller could reject disturbances and maintain precise bit positioning.

Key Results Achieved

  • The LQR controller successfully reduced drill bit oscillation from approximately ±15° to near 0° within 35 seconds of simulation time.
  • The system demonstrated excellent stability and strong disturbance rejection under modeled downhole vibration conditions.
  • Fast response and accurate positioning were achieved without excessive control effort.
  • The project proved that MATLAB/Simulink is a powerful, low-cost platform for designing and validating drilling control systems before physical implementation.
