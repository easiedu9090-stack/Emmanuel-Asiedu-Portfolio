SIMULATION & CONTROL SYSTEMS • 2022

Optimization of Bi-Directional Oil and Gas Drilling Equipment Using MATLAB-Based Modules

Final Year Project
All Nations University • October 2022

Project Overview

This project developed a MATLAB/Simulink simulation framework to optimize the positioning accuracy of bi-directional oil and gas drilling equipment by actively controlling downhole vibrations and shocks using a Linear Quadratic Regulator (LQR) controller.

Problem Statement

Drilling operations suffer from wellbore positioning uncertainty caused by downhole vibrations and shocks. These disturbances distort sensor signals (magnetometer and accelerometer), reducing the precision of the drill bit and wellbore placement. Traditional solutions often require expensive field testing.

Methodology & Technical Approach

  • Designed a high-fidelity drill string and bit model using mass-spring-damper principles in Simulink.
  • Developed sensor signal processing modules to filter noise from downhole vibrations.
  • Implemented a Linear Quadratic Regulator (LQR) controller using Root Locus and state-space methods.
  • Built a high-torque DC motor model integrated with the drill string system.
  • Used Bode plot analysis for controller tuning and stability verification.

Key Results

The LQR controller successfully damped oscillations and achieved precise positioning:

  • Reduced drill bit oscillation from ±15° to 0° within a 35-second simulation.
  • Controller effectively rejected disturbances caused by downhole vibrations.
  • System demonstrated good stability and fast response using LQR gain tuning.
Note: Detailed simulation results including Bode plots, drill string model diagrams, motor test results, and controlled bit position graphs are available in the full project report.
