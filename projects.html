SIMULATION & CONTROL • 2022

Optimization of Bi-Directional Oil and Gas Drilling Equipment Using MATLAB

Final Year Project • All Nations University, Koforidua

Project Overview

This project developed a MATLAB/Simulink simulation framework to optimize the positioning accuracy of bi-directional oil and gas drilling equipment by actively controlling downhole vibrations and shocks using a Linear Quadratic Regulator (LQR) controller. The goal was to improve wellbore placement precision using simulation instead of expensive field testing.

Aim

To develop a MATLAB/Simulink model that controls drilling string equipment to optimize system parameters and improve the accuracy of the drill bit positioning.

Objectives

  • To use MATLAB/Simulink to design and implement a control system strategy
  • To develop a drill string equipment model using Simulink
  • To develop a downhole vibration data filtering and control system
  • To design and test an LQR controller for precise drill bit positioning

Methodology

A high-fidelity drill string and bit model was developed using mass-spring-damper principles in Simulink. Sensor signal processing modules were created to filter noise from downhole vibrations. A Linear Quadratic Regulator (LQR) controller was designed using Root Locus and state-space methods. A high-torque DC motor model was integrated with the drill string system. The controller was tuned and validated through multiple simulation scenarios to achieve stable and accurate positioning under vibration disturbances.

Key Results Achieved

  • The LQR controller successfully damped oscillations caused by downhole vibrations.
  • Drill bit oscillation was reduced from ±15° to 0° within a 35-second simulation.
  • The system demonstrated good stability, fast response, and effective disturbance rejection.
  • The project proved that MATLAB/Simulink can serve as a low-cost and effective tool for drilling optimization and control system design.
