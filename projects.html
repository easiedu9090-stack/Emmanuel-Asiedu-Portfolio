SIMULATION & CONTROL • 2022

Optimization of Bi-Directional Oil and Gas Drilling Equipment Using MATLAB

Developed a complete MATLAB/Simulink simulation framework to optimize the positioning accuracy of bi-directional drilling equipment by actively controlling downhole vibrations and shocks using an LQR controller.
Key Contributions
  • → Designed LQR controller using Root Locus and state-space methods
  • → Built high-fidelity drill string model using mass-spring-damper approach
  • → Developed sensor signal processing modules in Simulink
  • → Achieved reduction of drill bit oscillation from ±15° to 0° in simulation
MATLAB & Simulink LQR Control State-Space Modeling Root Locus
